Parking Application

ParkingApp Repository on GitHub

This project is a concept for an app that could be used by students and faculty to make parking easier at Quinnipiac. It has features including space availability, automatic ticketing, viewing a database of all the tickets, etc. This was a group project where we used Git and GitHub for version control and collaberation. It was programmed using java and SQL.
